# 🚀 Project Title: Smart Farm 

Smart farm” is a website that intends to digitalize the farm. It provides different functionalities that pave way for the farmers to make it easier. This website aims on selling direct farm products without any agents or retailers so that farmers can get the maximum profit and customers can get farm fresh products. As to globalize this, other sellers can also sell their products through this website. The farmers can hire the workers that register through this website according to their necessity.
There are mainly 4 kinds of user admin, seller, customer, worker. Admin is the who has the whole control. He can add products, add news and can view customers, seller products etc. Sellers can add products, view admin products. Customers can purchase both products given by admin and seller. Workers can register to the website using login page and admins can view the worker registrations.

## 🌟 Key Features
* **Feature 1:** This system has four categories of admin, seller, worker and customer. Admin  can login to  the system for add product or update product details. admin can verify the orders and service requests and manage offers stock and payment and also can view the user info status. Admin can also add news into the site.
* **Feature 2:** Seller can login to this system to add products and can also view the products added by the admin. Seller can hire workers through this.
* **Feature 3:** Customer can add product to cart for buying products. He/She can add their personal and location details to this system. They can uses any type of payment method for transaction. (Net banking, Debit card, UPI etc.)
